## Blog Demo

This is a demo / template [click here](https://holgerjvogel.de) see what it could work. 

For now, It use vitepress & Vue & TypeScript to build.

Start & Test via execute command in a cmd

npm run-script build

npm run-script dev

check local [click here](  - localhost:3000)

## Highlights

- [x] pagination
- [x] dark mode
- [x] archives
- [x] tags
- [x] content level
- [x] comments
- [x] lastUpdated
- [x] github actions
- [x] mobile + desktop 

## Start

```shell
git clone
yarn
yarn dev
```
